# GenAI Contract Intelligence Demo

A password-protected Streamlit demo to extract structured rights from contracts, detect conflicts/holdbacks, export CSV/PDF reports, and generate Agile user stories.

---

## Features

- Password-protected login for demo access
- Upload your own PDF/DOCX contracts or use built-in sample contracts
- Structured rights extraction using OpenAI GPT
- Conflict & holdback detection
- Dashboard summary KPIs
- Export combined rights table as CSV or PDF
- Auto-generated user stories for each contract
